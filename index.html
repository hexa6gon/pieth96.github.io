Die Wahrheit über Twerion aka TwicePvP

Hi, ich bins Piet! Ich war bis vor kurzem noch Admin bei Twerion.
In diesem Text möchte ich euch kurz darüber informieren, warum das nicht mehr so ist.

Beginnen wir aber mal am Anfang. Ende November kam ich zu damals noch TwicePvP. Das Netzwerk war ein haufen Schrott, kein System, statische Server, es hat alles nur gelaggt.
Mecaanic war dort im Team und hatte mich gefragt, ob ich nicht Lust hätte das Netzwerk neu aufzubauen, ich entschied mich dafür.
Also habe ich so gut wie alles bis auf die Hauptserver (PvP, OpPvP usw.) neu entwickelt. Dazu zählten z.B. Bedwars und die allgemeine Netzwerkstruktur (Dynamische Gameserver usw.).
Ende Dezember wurde das Netzwerk dann unter dem Namen Twerion neu eröffnet, mit dem neuen System. Die Spielerzahlen haben sich in kurzer Zeit teilweise mehr als verdoppelt.
Hierzu muss gesagt sein, dass ich mein CloudSystem und den dazu gehörigen Masterserver (Eine Java Anwendung über die das ganze Netzwerk kommuniziert) obfuscated (Verschlüsselt) habe, um mir Sicherheiten zu schaffen.
Da ich aber persönlich eine schlechte Internetleitung besitze (2K), habe ich die meisten Anwendungen für Twerion über meinen BuildServer compilen lassen, damit ich die großen Dateien nicht jedes mal mit meiner Home-Leitung hochladen musste.
Damit verbunden ist natürlich ein Git Server auf dem der ganze Code gespeichert ist. (Ich habe hierzu Bitbucket und Bamboo von Atlassian benutzt).
Dieser Git und Bamboo Server waren natürlich auf meinem eigenem Root Server gehostet, dummerweise habe ich dort aber meine Twerion Email angegeben (Piet@Twerion.net).
Somit hatte der Besitzer von Twerion Zugriff auf meine Mail-Adresse und so gelang es ihm, über die Passwort vergessen Funktion Zugriff auf meinen Git Server und somit auch auf meinen ganzen Code von Twerion zu erlangen.
Das war dann in der Nacht zum 24.03.2016. Als ich dann auf der Arbeit auf mein Handy schaute, sah ich dass ich aus allen Gruppen entfernt wurde, natürlich mit der Frage warum.
Nach kurzer Recherche ist mir dann das mit dem Git-Server aufgefallen, er hatte nun den Source-Code und war der Meinung er brauche mich nicht mehr.
Ich wurde von vielen Personen gewarnt, als ich zu TwicePvP gegangen bin, diese Warnungen haben sich nun bestätigt. Anschließend möchte ich alles was ich für Twerion entwickelt habe veröffentlichen, unten ist eine Verlinkung zu meinem GitHub-Account. Bitte lest euch die Hinweise zu jeder Repo durch.

Mit freundlichen Grüßen
Piet
